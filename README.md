# 图书信息管理系统 python931

## 项目概述

图书信息管理系统是基于 Python 语言开发的一套文档与源代码集成的管理系统。它主要用于对图书馆的图书信息进行高效管理，实现图书信息的快速检索、借阅管理等功能。

## 技术栈

- **后端**: Python
- **数据库**: SQLite
- **前端**: HTML, CSS, JavaScript（可选）

## 功能模块

- **图书信息管理**
  - 图书信息录入
  - 图书信息编辑
  - 图书信息删除
- **检索系统**
  - 关键字搜索
  - 分类浏览
  - 高级检索
- **借阅管理**
  - 借阅记录
  - 还书处理
  - 超期提醒

## 系统角色

- 系统管理员：负责整个系统的维护与用户管理。
- 图书管理员：负责图书信息的录入、编辑和删除。
- 普通用户：可以进行图书检索和借阅。

## 运行环境

- 操作系统：兼容 Windows、Linux、MacOS
- Python 环境：Python 3.6 或更高版本
- 数据库：SQLite

## 部署步骤

1. 安装 Python 环境
2. 安装依赖库（详见 `requirements.txt`）
3. 初始化数据库（根据 `schema.sql`）
4. 运行系统（执行 `main.py`）

## 目录结构

```
图书信息管理系统
├── data                # 数据文件目录

```
